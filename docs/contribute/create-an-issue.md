# Create an issue

Create an issue when you:

- Found a bug
- Want to request a feature
- Have a question that needs a project decision

## Where

[Issues live here](https://github.com/Daniel-Gia/raspberry-pi-chromium-kiosk/issues)

## What to include

### For bugs

- Raspberry Pi model + OS version
- What you expected vs what happened
- Exact steps to reproduce
    - Logs (if relevant):
    - `systemctl status kiosk-browser`
    - `systemctl status admin-panel`
